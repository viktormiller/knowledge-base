    Docker Quick Start¶

    # Start container
docker run -d --name myapp -p 8080:80 nginx

# View logs
docker logs -f myapp

    Git Quick Commands¶

    # Quick commit
git add . && git commit -m "Update"

# Undo last commit (keep changes)
git reset --soft HEAD~1

    kubectl Essentials¶

    # Pod logs
kubectl logs -f pod-name

# Port forward
kubectl port-forward service/myapp 8080:80
